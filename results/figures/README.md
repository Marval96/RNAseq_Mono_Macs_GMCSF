# results/figures — Figuras

Esta carpeta contiene todas las figuras generadas por los scripts de análisis.

## Convención de nombres

```
{número_paso}_{tipo_figura}_{descripción}.{ext}
```

**Ejemplos:**
- `04_PCA_monocyte_vs_macrophage.pdf`
- `04_volcano_monocyte_vs_macrophage.pdf`
- `04_heatmap_top50_DEGs.pdf`
- `05_GO_BP_dotplot.pdf`

## Formatos

- **PDF** — para publicación (vectorial, alta calidad)
- **PNG** — para presentaciones / vista rápida

> ⚠️ Las figuras se regeneran ejecutando los scripts en `scripts/`. No se modifican manualmente.
